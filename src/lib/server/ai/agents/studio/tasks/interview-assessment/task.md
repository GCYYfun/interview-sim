# 面试评估任务指南

您需要完成一个完整的候选人面试评估任务。

## 数据获取

首先，使用提供的Tool从选中的数据源获取必要信息：
- **简历内容**：通过sourceId获取候选人的简历和背景信息
- **面试对话**：获取面试过程中的完整对话记录
- **职位描述**：获取目标职位的详细要求和责任

## 分析维度

对每个维度进行深入分析：

1. **技术匹配度** (40%)
   - 对比职位要求和候选人技能
   - 指出核心技能的匹配和缺陷
   - 评估学习新技能的能力

2. **文化匹配度** (30%)
   - 分析沟通风格和团队适应能力
   - 评估价值观契合度
   - 判断合作态度

3. **经验水平** (20%)
   - 评估从业年限的相关性
   - 分析职业发展的连贯性
   - 考察项目经验的复杂度

4. **动力与成长** (10%)
   - 评估对职位的真实兴趣
   - 判断长期职业规划
   - 预测团队贡献

## 生成报告

生成结构化的评估报告，包括：
- 总体推荐（强力推荐/推荐/保留/不推荐）
- 各维度分数 (X/10) 和详细分析
- 关键优势（3-5条，附证据）
- 关注事项（任何风险或不确定性）
- 面试亮点和建议

## 输出要求

- 使用清晰的markdown格式
- 从来源中引用具体证据
- 保持客观和专业的语气
- 提供可行的建议供后续决策
