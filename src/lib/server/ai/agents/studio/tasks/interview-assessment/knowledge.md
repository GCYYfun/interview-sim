# 面试评估知识库 - 招聘基础

本知识库提供了招聘评估的最佳实践和标准化方法。

## 核心原则

1. **证据为基础**：所有评论都必须有可观察的证据支持
2. **结构清晰**：明确区分优势、风险和知识空白
3. **专业态度**：保持专业的招聘导向的语气
4. **可读性强**：使用清晰的markdown格式便于工作区阅读

## 评估要点

### 技术匹配度
- 检查候选人是否掌握职位所需的必要技能
- 比较候选人的技能深度和广度
- 评估是否有明显的技能缺陷

### 文化匹配度
- 观察候选人的沟通风格和表达能力
- 评估解决问题的思维方式
- 判断团队协作意愿和适应能力

### 经验水平
- 复审职业发展路径的连贯性
- 评估处理过的项目复杂度
- 判断行业经验的相关性

### 动力与成长
- 评估对职位和公司的真实兴趣
- 判断学习和成长的积极性
- 预测长期职业发展方向

## 报告标准

清晰的结构化报告应包含：
- 整体推荐意见
- 各维度详细评分和分析
- 关键优势和风险
- 面试亮点和需要关注的地方
- 最终建议和后续行动
